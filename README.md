TareaHilos 

Proyecto desarrollado en **Java** para practicar el manejo de hilos (concurrencia). Ideal para entender cómo ejecutar múltiples tareas simultáneamente.

Tecnologías utilizadas
- **Lenguaje:** Java
- **Herramientas:** VS Code

El código principal se encuentra en la ruta: `src/TareaHilos`
